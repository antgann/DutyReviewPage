Shakemap Manual Execution Instructions

To manually process an event or scenario:

On primary system, this will PUBLISH the event to outside

1. Check http://rift.gps.caltech.edu/productstatus/statuslights.php to see which system is primary

2. Login to the primary system

3. cd /home/shake/bin

4. run script ./shake -event event_id

If run from Primary, shakemaps will appear on

http://www.cisn.org/shakemap/sc/shake/

http://earthquake.usgs.gov/earthquakes/shakemap/list.php

NOTE: this may trigger other alarms or email lists to the public!
